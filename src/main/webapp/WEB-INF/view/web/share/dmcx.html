机构日志

摩米创新工场是以电子科技大学专业技术团队为依托，立足光电信息技术创新驱动，集成技术成果转移、项目孵化器、企业加速器、产业聚集园于一体的创新服务平台。2013年8月电子科技大学刘霖博士与浙江省宁波市鄞州区政府签署“摩米创新工场引进协议”，宁波摩米创新工场电子科技有限公司与2013年9月成立，注册资本人民币2000万元。
摩米创新工场广集科技成果、技术服务、工业设计、孵化场地、中试环境、人才资源、融资渠道、政府扶持等资源，实现工业现场、创意设计、网络推广、人才培训、法律顾问、财务代理、金融投资等专业服务，并与企业密切合作，共建企业联合实验室、企业工场技术中心，快速推动企业集群，整合产业链资源，加速产业升级转型，成就企业家梦想。
